# Neural Network Playground

## Development Environment
```
Ubuntu 20.04
Python 3.12.3
Pytorch 2.3.1
Cuda 12.1
```

## Setup

```bash
git clone https://github.com/MizuhoAOKI/neural_network.git
cd neural_network
rye sync
```

## Usage

```bash
rye run jupyter notebook notebooks/mlp_pytorch.ipynb
```
